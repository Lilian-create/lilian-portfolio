# Lilian Portfolio

A modern, interactive React portfolio website showcasing media production, content strategy, and translation services.

## Features

- ✨ Smooth animations with Framer Motion
- 🎨 Beautiful gradient design with Tailwind CSS
- 📱 Fully responsive design
- ⚡ Fast performance with React
- 🎯 Scroll-triggered animations
- 🧲 Interactive magnetic button effects

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Framer Motion** - Animation library
- **Tailwind CSS** - Utility-first CSS
- **Lucide React** - Icon library

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Lilian-create/lilian-portfolio.git

# Navigate to project directory
cd lilian-portfolio

# Install dependencies
npm install

# Start development server
npm start
```

The application will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build/` directory.

## Project Structure

```
src/
├── App.tsx           # Main portfolio component
├── index.tsx         # React entry point
└── index.css         # Global styles
```

## Sections

1. **Hero** - Introduction with animated gradient text and profile image
2. **Marquee** - Scrolling badges of skills and tools
3. **About** - Background information with scroll animations
4. **Services** - 4 core capabilities
5. **Projects** - Sticky scroll project showcase
6. **Footer** - Social links and contact

## Deployment

This portfolio can be deployed to:
- **Vercel** (recommended for Next.js or React)
- **GitHub Pages**
- **Netlify**
- **AWS S3 + CloudFront**

### Deploy to GitHub Pages

```bash
npm run build
# Push the build folder to gh-pages branch
```

### Deploy to Vercel

```bash
npm install -g vercel
vercel
```
